# Hybrid Face Recognition Attendance System

A real-time attendance system using face recognition with a hybrid
local + API fallback architecture.

## Features
- SCRFD face detection (InsightFace)
- ArcFace embeddings with FAISS
- Multi-angle registration
- MySQL + Daily CSV attendance
- Face++ API fallback
- Modular & GUI-ready architecture

## Tech Stack
- Python 3.x
- OpenCV
- InsightFace (SCRFD + ArcFace)
- FAISS
- MySQL
- React (frontend – in progress)
